FireGB v1.0

Install Instructions:
Install sdl & sdl_ttf through your package manager

make
su
make install

Run instructions:
fgb romName

Keys:
Arrow Keys
A: Z 
B: X
Start: Enter, A
Select: Backspace, S
Speed Mode: Space Bar
Fullscreen: F
Scale Screen: 1, 2, 3, 4, 5, 6, 7, 8
Mute: m
Have fun!
