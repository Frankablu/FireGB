FireGB v0.9 Mac

Install Instructions:
Install SDL from http://www.libsdl.org/download-1.2.php if not already present

chmod +x macosxinstall.sh
./macosxinstall.sh

Run Instructions:
fgb romName

Keys:
Arrow Keys
A: Z 
B: X
Start: Enter, A
Select: Backspace, S
Speed Mode: Space Bar
Fullscreen: F
Scale Screen: 1, 2, 3, 4, 5, 6, 7, 8
Mute: m
Have fun!

Build Instructions:
Install XCode
Install SDL
Open FireGBXCode.xcodeproj
Press the Green Arrow
Copy the Binary into /usr/bin/
