FireGB Emulator v1.0

You may need to install vcredist_x86.exe (included) first

Usage: Run FireGB.exe

Click to bring up the menubar

Keys:
Arrow Keys
A: Z 
B: X
Start: Enter, A
Select: Backspace, S
Speed Mode: Space Bar
Fullscreen: F
Scale Screen: 1, 2, 3, 4, 5, 6, 7, 8
Mute: m
Have fun!

This program uses the SDL, SDL_gfx & SDL_ttf libraries
